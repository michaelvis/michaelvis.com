# michaelvis.com

Personal website for Michael Vis — Electrical & Computer Engineer.

Built with plain HTML and CSS. No frameworks, no build step.

## Pages

- **About** — Bio, photo, and social links
- **Projects** — Card-based project showcase
- **Music** — Recordings and sheet music
- **Contact** — Email and socials

## Local Development

```bash
python3 -m http.server 3000
```

Then open `http://localhost:3000` in your browser.

## Adding Content

- **New project:** Duplicate a `.project-card` block in `projects.html`
- **New video/audio:** Duplicate a `.video-card` block in `music.html`
- **Styles:** All shared styles are in `style.css`
